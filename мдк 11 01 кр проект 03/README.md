# PHP Auth Portfolio

Простой сайт-портфолио с авторизацией, регистрацией по номеру телефона и паролю и восстановлением пароля. Хранение данных реализовано через SQLite (`database.db`). Запуск осуществляется с помощью Node.js, который поднимает встроенный PHP‑сервер.

## Требования

- Node.js 18+
- PHP 8+

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Сервер поднимется на `http://127.0.0.1:8000`. Все страницы (`login.php`, `register.php`, `reset_password.php`, `index.php`, `logout.php`) лежат в директории `public/`.

## Структура

- `server.js` — запускает встроенный PHP-сервер через Node.js.
- `public/` — PHP-страницы и вспомогательные скрипты.
- `database.db` — локальная база данных SQLite (создаётся автоматически при первом обращении).
- `sms.log` — журнал SMS-кодов (заглушка вместо реальных отправок).

## Подтверждение смены пароля

1. Пользователь вводит номер телефона и новый пароль.
2. Генерируется одноразовый код (6 цифр), он логируется в `sms.log`. Для продакшена замените запись в файл интеграцией с SMS-провайдером.
3. Пароль обновляется только после успешного ввода кода, что исключает смену без подтверждения по телефону.

## GitHub

Перед публикацией убедитесь, что файл `database.db` добавлен в `.gitignore`, а в репозитории нет персональных данных.
