# Личное портфолио с аутентификацией

Одностраничное приложение с регистрацией, входом, восстановлением пароля в три шага и страницей портфолио. Архитектура: Node.js (прокси) + PHP (логика) + SQLite.

## Стек
- Node.js 14+, Express, http-proxy-middleware  
- PHP 7.4+ (встроенный сервер)  
- SQLite3  
- Ванильные HTML/CSS/JS

## Установка и запуск
```bash
# переход в директорию проекта
cd "мдк 11 01 проект 07"

# установка Node-зависимостей
npm install

# инициализация базы
php php/init_db.php

# старт (Node поднимет PHP-сервер и проксирует)
npm start
```
Откройте `http://localhost:3000`.

## Структура
- `index.php` — редирект на `php/login.php`
- `php/` — регистрация, вход, восстановление, портфолио, конфиг, init DB
- `css/styles.css` — стили, адаптив, анимации
- `js/script.js` — маска телефона, блокировка русских символов, показ пароля по удержанию, плавная прокрутка, анимации
- `js/server.js` — Express-прокси + запуск PHP builtin
- `database.db` — SQLite (создаётся скриптом)

## Основные возможности
- Регистрация с проверкой уникальности телефона, хеширование паролей (`password_hash`)
- Вход, защита CSRF, сессии только через cookies (HttpOnly)
- Восстановление пароля: запрос кода (6 цифр, срок 15 минут, 3 запроса/15 мин), верификация (3 попытки), смена пароля
- Портфолио доступно только авторизованным; отображается телефон текущего пользователя
- UI: sticky навбар, плавная прокрутка, hover-эффекты, адаптив под desktop/tablet/mobile
- Клиентские проверки: маска телефона `+7-xxx-xxx-xx-xx`, блокировка русских символов в пароле (клавиатура/вставка/IME), показ пароля при удержании

## Конфигурация
- Порты: `PORT=3000`, `PHP_PORT=8000` (env)
- Настройки безопасности: см. `php/config.php` (`SESSION_DURATION`, `RECOVERY_CODE_LENGTH`, `RECOVERY_CODE_EXPIRY`)

## Примечания
- Демонстрационный режим восстановления выводит код на страницу (для продакшена заменить на SMS и убрать вывод).
- В продакшене включайте HTTPS и `secure` для cookies.
