# Адаптивный лендинг по макету Figma

Готовая вёрстка с mobile-first подходом, интерактивным слайдером и Node.js сервером на Express. Все изображения подготовлены в PNG, фон задаётся из `public/assets/`.

## Требования

- Node.js 18+

## Структура

- `server.js` — Express-сервер, раздаёт статику из `public/`, записывает сообщения формы в `contact-messages.log`.
- `public/index.html` — главная страница с навигацией, hero-секцией, контентными блоками и слайдером.
- `public/styles.css` — mobile-first стили с брейкпоинтами под телефон / планшет / десктоп.
- `public/scripts.js` — плавный скролл, бургер-меню, смена стиля шапки, слайдер с автопрокруткой и свайпами, отправка формы.
- `public/assets/` — PNG-изображения (фон, контент, слайды).

## Запуск

```bash
npm install
npm start
```

Открыть `http://127.0.0.1:8000/`.

## Проверка функционала

- Меню фиксировано, меняет фон при прокрутке, плавно скроллит к секциям, на мобильных раскрывается бургером.
- Слайдер листается кнопками, индикаторами, свайпами, работает автопрокрутка с паузой.
- Footer показывает текущий год, контактная форма отправляет данные на `/api/contact` (сохраняются в лог).
