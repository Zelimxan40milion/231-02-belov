# Система автоматизированного тестирования форм аутентификации

Комплекс решает задачу проверки регистраций, входа и восстановления пароля с русскоязычным веб-интерфейсом запуска тестов.

## Возможности
- 50 автоматических тестов, распределённых по группам (регистрация, вход, восстановление).
- Изолированная тестовая база SQLite, пересоздаётся для каждого теста.
- Автоматический пропуск тестов, выполняющихся дольше 1 секунды.
- Проверка валидных и невалидных сценариев и валидаций полей.
- Детальная статистика: длительность, статус, количество успехов/ошибок/пропусков.
- Веб-интерфейс на Flask с запуском тестов и просмотром результатов (русский язык).

## Быстрый старт
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите CLI-раннер:
   ```bash
   python run_tests.py
   ```
   Результаты сохраняются в `reports/last_results.json`.
3. Поднимите веб-интерфейс:
   ```bash
   flask --app web.server run --reload
   ```
   Перейдите на `http://127.0.0.1:5000/`.

## Структура
- `app/auth_logic.py` — бизнес-логика форм аутентификации.
- `app/db.py` — управление тестовой БД SQLite.
- `app/test_runner.py` — фреймворк тестов и статистики.
- `tests/test_scenarios.py` — 50 сценариев.
- `web/server.py` + `web/templates` — интерфейс запуска/просмотра.
- `run_tests.py` — автономный запуск.

## Примечания
- Все строки интерфейса на русском.
- При запуске через веб-интерфейс тесты выполняются синхронно, чтобы гарантировать корректное отображение статистики.
