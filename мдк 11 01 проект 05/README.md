## Система автотестов аутентификации (PHP + PHPUnit)

Проект реализует комплексную систему автоматического тестирования форм аутентификации (регистрация, вход, восстановление пароля) с:

- изолированной тестовой БД SQLite;
- собственным тестовым раннером (`run_tests.php`) с таймаутом 1 сек;
- дополнительной структурой тестов на **PHPUnit**;
- веб-интерфейсом (`index.php`) для запуска тестов и просмотра результатов на русском языке.

### Установка

1. Установите PHP 8.1+ и Composer.
2. В корне проекта выполните:

```bash
composer install
```

### Запуск тестов

- Собственный раннер с таймаутом:

```bash
php run_tests.php
```

- PHPUnit (структура `tests_phpunit/`):

```bash
vendor/bin/phpunit
```

Тесты разделены по группам:

- `tests_phpunit/Validation` – 10 тестов валидации;
- `tests_phpunit/Registration` – 15 тестов регистрации;
- `tests_phpunit/Login` – 10 тестов входа;
- `tests_phpunit/PasswordReset` – 15 тестов восстановления пароля.

### Веб-интерфейс результатов

1. Запустите встроенный сервер PHP:

```bash
php -S localhost:8000
```

2. Откройте браузер и перейдите по адресу `http://localhost:8000/`.

Страница `index.php` предоставляет:

- кнопку запуска полного набора тестов;
- цветовую индикацию статусов;
- детальную статистику выполнения;
- информацию по каждому тесту (группа, имя, статус, время, сообщение).
